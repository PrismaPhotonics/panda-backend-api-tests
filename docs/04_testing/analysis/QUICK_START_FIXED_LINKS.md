3. **עבור ל-Cursor**
4. **Ctrl+P** → **Ctrl+V** → **Enter**

## 🐛 **פתרון בעיות:**

### **בעיה: "Copy Path" לא עובד**

**פתרון:**
- בדוק שה-JavaScript מופעל בדפדפן
- נסה דפדפן אחר (Chrome מומלץ)
- או: העתק את הנתיב ידנית מהתיבה הירוקה

### **בעיה: Ctrl+P לא פותח דיאלוג ב-Cursor**

**פתרון:**
- ודא ש-Cursor פעיל (לא מינימלי)
- נסה Ctrl+Shift+P (Command Palette)
- או: לחץ על תפריט File → Go to File

### **בעיה: הקובץ נפתח אבל לא בשורה הנכונה**

**פתרון:**
- הנתיב כולל `:395` בסוף
- אם Cursor לא קופץ לשורה, לחץ Ctrl+G
- הקלד את מספר השורה ולחץ Enter
