🔧 איך לעדכן את הקונפיג של PandaApp
═══════════════════════════════════════════════════════════════

⚠️ הבעיה:
────────
האפליקציה מנסה להתחבר ל-localhost:3000 במקום לסביבה האמיתית!

✅ הפתרון (בחר אחת מהאופציות):
═══════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────┐
│ אופציה 1: הרצת סקריפט אוטומטי (מומלץ! ⭐)                 │
└─────────────────────────────────────────────────────────────┘

1. סגור את PandaApp (חשוב!)

2. מצא את הקובץ:
   📄 C:\Users\roy.avrahami\Downloads\Update_PandaApp_Config.ps1

3. לחיצה ימנית על הקובץ → "Run with PowerShell"
   (אם מבקש Admin - אשר!)

4. הסקריפט יעשה הכל אוטומטית:
   ✅ יגבה את הקובץ הישן
   ✅ יעדכן את הקונפיג החדש
   ✅ ישאל אם לפתוח את האפליקציה

5. תסכים לפתוח את PandaApp

6. בדוק בלוגים שהכתובת השתנתה!


┌─────────────────────────────────────────────────────────────┐
│ אופציה 2: עדכון ידני                                      │
└─────────────────────────────────────────────────────────────┘

1. סגור את PandaApp

2. פתח File Explorer והדבק:
   C:\Program Files\Prisma\PandaApp

3. גבה את usersettings.json (שנה שם ל-usersettings.json.old)

4. העתק את הקובץ:
   C:\Users\roy.avrahami\Downloads\usersettings.cleaned.json
   
   לתוך:
   C:\Program Files\Prisma\PandaApp\
   
   ושנה שם ל: usersettings.json

5. אם Windows מבקש הרשאות Admin - אשר

6. פתח את PandaApp מחדש
