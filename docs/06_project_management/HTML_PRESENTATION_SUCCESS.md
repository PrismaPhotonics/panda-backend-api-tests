## 🐛 **פתרון בעיות:**

### **בעיה: לינקים לא פותחים את Cursor**

**פתרון 1:** הרשאת דפדפן
- הדפדפן עשוי לחסום פרוטוקול `vscode://`
- אפשר כאשר מופיע הדיאלוג
- סמן "זכור את הבחירה"

**פתרון 2:** פתיחה ידנית
- אם הלינק נכשל, שים לב לנתיב הקובץ בשקופית
- לחץ Ctrl+P ב-Cursor
- הדבק את נתיב הקובץ
- לחץ Enter

**פתרון 3:** נסה דפדפן אחר
- נסה Chrome, Edge, או Firefox
- חלק מהדפדפנים מטפלים בפרוטוקולים טוב יותר

### **בעיה: המצגת נראית מוזר**

**פתרון:** נקה מטמון דפדפן
- לחץ Ctrl+F5 לרענון מלא
- או: Ctrl+Shift+Delete → נקה מטמון

### **בעיה: השקופיות לא זזות**

**פתרון:** בדוק JavaScript
- ודא ש-JavaScript מופעל בדפדפן
- נסה דפדפן אחר
- טען מחדש את הדף (F5)
