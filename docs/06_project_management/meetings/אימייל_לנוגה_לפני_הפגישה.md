## 🎯 מה קרה?

השבוע רצנו **בדיקות אוטומציה** על ה-API של ה-Backend (Focus Server).

**התוצאה המפתיעה:**
- ✅ 21/21 בדיקות עברו = השרת **מקבל הכל**
- 🔴 5/5 בדיקות requirement נכשלו = **אין ווולידציה** בצד השרת

---

## 🔍 מה מצאנו? (4 בעיות)

### 1. **displayInfo.height** - מקבל ערכים שליליים
```json
{"displayInfo": {"height": -100}}  → ✅ 200 OK  // ⚠️
```

### 2. **nfftSelection** - מקבל ערכים לא תקינים
```json
{"nfftSelection": 1000}  → ✅ 200 OK  // ❌ לא power-of-2
{"nfftSelection": 4096}  → ✅ 200 OK  // ❌ מעל 2048
```

### 3. **frequencyRange** - אין בדיקת Nyquist
```json
{"frequencyRange": {"max": 600}}  → ✅ 200 OK  
// ❌ אם PRR=1000, Nyquist=500, אז 600 לא תקין!
```

### 4. **channels** - מקבל מעל 2500
```json
{"channels": {"min": 1, "max": 2501}}  → ✅ 200 OK  // ❌ מעל הגבול
```
