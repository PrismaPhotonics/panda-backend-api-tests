## 📊 סיכום מהיר - 4 הבעיות

| # | בעיה | דוגמה | החלטה |
|---|------|-------|-------|
| 1️⃣ | **height** שלילי | `-100` → OK | צריך לדחות |
| 2️⃣ | **nfft** לא תקין | `1000` → OK | צריך dropdown |
| 3️⃣ | **freq** > Nyquist | `600>500` → OK | צריך dynamic |
| 4️⃣ | **channels** > 2500 | `2501` → OK | צריך limit |
